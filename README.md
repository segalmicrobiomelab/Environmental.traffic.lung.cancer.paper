**R Code Repository for Data Analysis for Lung Cancer and Traffic PM2.5 paper**


**Prerequisites:**

MacOS/Windows/Linux
R and RStudio

This repository contains code and data used for epidemiologic, human BAL transcriptomic, and mouse model lung transcriptomic analyses related to lung cancer and traffic vs. non-traffic PM2.5 exposure.

**The repository is organized into three main folders:**

SEER/ – SEER epidemiologic analyses (R scripts + datasets)

HUMAN/ – Human lung BAL transcriptomic analyses (R scripts + datasets)

MOUSE/ – Mouse lung transcriptomic analyses (R scripts + datasets)


Each folder includes: R ode for the analysis and the dataset(s) needed to reproduce results.
